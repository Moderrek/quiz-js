W Obiektowym Modelu Dokumentu obiektem stojącym najwyżej w hierarchii jest obiekt:
document
innerHTML
window
Zdarzenie w skrypcie HTML umieszcza się:
jako atrybut o nazwie zdarzenia w znaczniku, którego zdarzenie dotyczy
jako atrybut o nazwie zdarzenia w znaczniku &ltSCRIPT&gt
jako oddzielny znacznik o nazwie zdarzenia
Aby kilka kontrolek typu radio stanowiły jedną grupę należy każdej z nich:
nadać atrybut ID o identycznej nazwie
nadać atrybut NAME o identycznej nazwie
nadać atrybut VALUE o identycznej nazwie
Kontrolka tekstowa, w której podczas wpisywania znaki są zastępowane kropkami jest typu:
hidden
password
pwd
Kontrolka mająca kształt przesuwalnego suwaka jest typu:
range
ruler
slider
Które polecenie spowoduje pobranie do zmiennej a zawartości z pola tekstowego login:
a = document.getElementById('login')
a = document.getElementById('login').id
a = document.getElementById('login').value
Czy w nazwach zmiennych w języku JavaScript są rozróżniane wielkie i małe litery?
nie
tak
zależnie od używanego systemu operacyjnego

def
fun 
function

[test_logiczny] ? [wyrażenie_1] : [wyrażenie_2]
